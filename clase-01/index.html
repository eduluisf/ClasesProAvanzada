Listas

Colecciones ordenadas y mutables de datos

Listas

¿Qué es una lista?

Definición

  • Colección ordenada de elementos
  • Puede contener cualquier tipo de dato
  • Mutable — sus elementos se pueden cambiar
  • Acceso por índice, empieza en 0

Índices clave

  • lista[0] → primer elemento
  • lista[-1] → último elemento
  • lista[1:3] → sublista (slice)
# Crear una lista
frutas = ["manzana", "pera", "uva"]

# Lista mixta
mixta = [1, "hola", 3.14, True]

# Acceder a elementos
print(frutas[0])    # manzana
print(frutas[-1])   # uva
print(frutas[0:2])  # ['manzana', 'pera']

# Modificar un elemento
frutas[1] = "mango"
print(frutas)  # ['manzana', 'mango', 'uva']

Métodos esenciales

nums = [3, 1, 4, 1, 5]

nums.append(9)
print(nums)   # [3, 1, 4, 1, 5, 9]

nums.insert(0, 100)

print(len(nums))     # 6
print(nums.count(1)) # 1
Módulo 02

Ciclo For

Itera sobre cualquier secuencia de forma clara y directa

Ciclo For

For básico y range()

¿Qué es range()?

  • range(n) genera 0, 1, 2 … n-1
  • range(a, b) genera a, a+1 … b-1
  • range(a, b, paso) salta de paso en paso
# Iterar una lista directamente
frutas = ["manzana", "pera", "uva"]
for f in frutas:
    print(f)
# manzana / pera / uva
# range(5) → 0, 1, 2, 3, 4
for i in range(5):
    print(i)

# range(2, 8) → 2, 3, 4, 5, 6, 7
for i in range(2, 8):
    print(i)

# range(0, 10, 2) → 0, 2, 4, 6, 8
for i in range(0, 10, 2):
    print(i)
Ciclo For — Python vs Java

Recorrer una lista por posición

Python
frutas = ["manzana", "pera", "uva"]

# range(len(frutas)) → 0, 1, 2
for i in range(len(frutas)):
    print(i, frutas[i])
# 0 manzana
# 1 pera
# 2 uva

range() = índices de la lista

range(len(lista)) genera exactamente los índices 0, 1, 2 … n-1, igual que Java con i < arr.length

Java
String[] frutas = {"manzana", "pera", "uva"};

// i empieza en 0, sube 1 a 1
for (int i = 0; i < frutas.length; i++) {
    System.out.println(i + " " + frutas[i]);
}
// 0 manzana
// 1 pera
// 2 uva

Son equivalentes

  • Java: i = 0; i < arr.length; i++
  • Python: i in range(len(lista))
  • Ambos recorren los índices 0 → n-1
Ciclo For

Variantes útiles

enumerate() — índice + valor

frutas = ["manzana", "pera", "uva"]
for i, f in enumerate(frutas):
    print(i, f)
# 0 manzana / 1 pera / 2 uva

zip() — dos listas en paralelo

nombres = ["Ana", "Luis"]
notas   = [9, 7]
for n, nota in zip(nombres, notas):
    print(n, nota)   # Ana 9 / Luis 7

List comprehension — for en una línea

# Cuadrados del 0 al 4
cuadrados = [x**2 for x in range(5)]
print(cuadrados)  # [0, 1, 4, 9, 16]

# Solo los pares
pares = [x for x in range(10) if x % 2 == 0]
print(pares)  # [0, 2, 4, 6, 8]

for … else

for n in [1, 2, 3]:
    print(n)
else:
    print("Ciclo terminó sin break")
Módulo 03

Ciclo While

Se repite mientras una condición sea verdadera

Ciclo While

While básico

¿Cuándo usarlo?

  • No sabes cuántas veces va a iterar
  • Ideal para validar input del usuario
  • La condición de salida depende del proceso
  • Puede ser infinito → usar break
# Contador básico
contador = 0
while contador < 5:
    print(contador)
    contador += 1
# 0  1  2  3  4
# Pedir dato hasta que sea válido
edad = -1
while edad < 0:
    edad = int(input("Ingresa tu edad: "))
print("Edad válida:", edad)

Diferencia clave con For

  • for → cantidad conocida de iteraciones
  • while → cantidad desconocida, depende de condición
Ciclo While

break y continue

break — sale del ciclo inmediatamente

i = 0
while True:       # loop infinito
    if i == 5:
        break     # sale aquí
    print(i)
    i += 1
# imprime: 0 1 2 3 4

Regla práctica

  • break → termina el ciclo
  • continue → salta esta iteración

continue — salta al siguiente ciclo

i = 0
while i < 6:
    i += 1
    if i % 2 == 0:
        continue  # salta los pares
    print(i)
# imprime: 1  3  5
# También funciona con for
for x in range(5):
    if x == 3:
        break
    print(x)  # 0 1 2
Ciclos

For vs While

for — úsalo cuando…

  • Sabes cuántas veces va a iterar
  • Recorres una lista, string o rango
  • Necesitas el índice con enumerate
  • Quieres iterar en paralelo con zip

while — úsalo cuando…

  • No sabes cuántas iteraciones habrá
  • Esperas input del usuario
  • La condición de salida depende del proceso
  • Necesitas un bucle infinito con break

Regla general

Si puedes expresar la condición como un rango o iterable concreto → usa for. Si la condición de parada es dinámica → usa while.

Módulo 04

Funciones

Bloques de código reutilizables que nos ayudan a organizar la lógica

Funciones

Sin return

Características

  • Realizan acciones (imprimir, modificar)
  • No devuelven un valor útil
  • Retornan None de forma implícita
  • Se llaman por sus efectos secundarios
def saludar(nombre = "mundo"):
    print(f"¡Hola, {nombre}!")

saludar()          # ¡Hola, mundo!
saludar("Ana")    # ¡Hola, Ana!
def mostrar_lista(lista):
    print("Elementos:")
    for i, elem in enumerate(lista):
        print(f"  [{i}] {elem}")

mostrar_lista(["Python", "Java"])
# Elementos:
#   [0] Python
#   [1] Java

# Sin return → el valor es None
resultado = saludar("Luis")
print(resultado)  # None
Funciones

Con return

Características

  • Devuelven un valor al llamador
  • Podemos guardar o usar el resultado
  • El código tras return no se ejecuta
  • Pueden retornar múltiples valores
def sumar(a, b):
    return a + b

resultado = sumar(3, 4)
print(resultado)           # 7
print(sumar(10, 5) * 2)  # 30
# Retornar múltiples valores
def min_max(lista):
    return min(lista), max(lista)

menor, mayor = min_max([3, 1, 9, 2])
print(menor, mayor)  # 1  9

Diferencia clave

  • Sin return → solo imprime, valor es None
  • Con return → podemos usar el valor en cálculos
# Con return → usable en expresiones
def doble(x):
    return x * 2

print(doble(3) + doble(4))  # 14

Ejercicio 1 — Listas
Solución

Gestionar una lista de compras

compras = ["leche", "pan", "huevos"]

# Agregar 2 productos
compras.append("arroz")
compras.append("café")

# Eliminar el primero
compras.pop(0)

# Ordenar
compras.sort()

print(compras)
# ['arroz', 'café', 'huevos', 'pan']

Puntos clave

  • pop(0) elimina y devuelve el índice 0
  • sort() ordena la misma lista (in-place)
  • El orden final: ["arroz", "café", "huevos", "pan"]

Suma de números pares

Objetivo

Usa un ciclo for para sumar todos los números pares del 1 al 100 e imprimir el resultado.

Pista

  • Usa range() con el paso correcto ó
  • Usa if n % 2 == 0 dentro del for
  • Resultado esperado: 2550

Intenta resolverlo
antes de pasar al siguiente slide

Ejercicio 2 — Ciclo For
Solución

Suma de números pares

Opción A — paso en range
total = 0
for n in range(2, 101, 2):
    total += n

print("Suma:", total)  # 2550
Opción B — condicional dentro del for
total = 0
for n in range(1, 101):
    if n % 2 == 0:
        total += n

print("Suma:", total)  # 2550

¿Por qué range(2, 101, 2)?

Empieza en 2 (primer par), termina antes de 101 (incluye 100), salta de 2 en 2 → solo números pares.

Enunciado

Adivina el número

Objetivo

Crea un juego donde el usuario tiene que adivinar el número secreto 42. Usa un while que siga pidiendo intentos hasta que acierte. Muestra si debe ir más alto o más bajo.

Ejercicio 3 — Ciclo While
Solución

Adivina el número

secreto = 42
intento = 0

while intento != secreto:
    intento = int(input("Adivina: "))

    if intento < secreto:
        print("Más alto ↑")
    elif intento > secreto:
        print("Más bajo ↓")

print("¡Correcto!")

¿Por qué while aquí?

  • No sabemos cuántos intentos hará el usuario
  • La condición depende del valor ingresado
  • Podría resolverlo en 1 intento o en 100
  • Un for requeriría saber el número de vueltas

Ejercicio 4 — Función sin return
Solución

Tabla de multiplicar

def tabla(n):
    print(f"--- Tabla del {n} ---")
    for i in range(1, 11):
        print(f"{n} x {i} = {n * i}")

tabla(3)
tabla(7)

Puntos clave

  • La función imprime pero no retorna
  • range(1, 11) genera 1, 2, … 10
  • Podemos reutilizar la función con cualquier n
  • f-string permite formatear fácilmente

Enunciado

Promedio de una lista

Crea una función promedio(lista) que reciba una lista de números y retorne su promedio. Si la lista está vacía, retorna 0.

Extra

Usa el valor retornado en una expresión para verificar si el promedio es mayor o igual a 6 (aprobado).

Intenta resolverlo
antes de pasar al siguiente slide

Ejercicio 5 — Función con return
Solución

Promedio de una lista

def promedio(lista):
    if len(lista) == 0:
        return 0
    return sum(lista) / len(lista)

notas = [8, 9, 7, 10, 6]
p = promedio(notas)

print(f"Promedio: {p}")      # 8.0
print(f"Aprobado: {p >= 6}") # True

print(promedio([]))  # 0

¿Por qué return aquí?

  • El valor se usa en expresiones: p >= 6
  • Podemos pasarlo a otras funciones
  • Sin return solo podríamos imprimirlo

Dos returns

  • El primero cubre el caso especial (lista vacía)
  • El segundo retorna el cálculo normal
  • Python ejecuta el primero que encuentra
Resumen

Lo que aprendimos

Listas

  • Crear y acceder por índice
  • append, pop, sort
  • Slicing [a:b]
  • Índices negativos

Ciclos

  • for + range, enumerate, zip
  • Recorrer por posición (≡ Java)
  • while con condición dinámica
  • break, continue

Funciones

  • def nombre(params)
  • Sin return → efectos / imprime
  • Con return → valor reutilizable
  • Parámetros por defecto
